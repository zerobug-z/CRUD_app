# Lab Sample Management System

Railway-URL: https://crudapp-production-db8f.up.railway.app/swagger-ui/index.html

This application helps manage laboratory samples by allowing users to add, update, track, and delete sample records. It also stores detailed sample information to improve data monitoring and organization within the lab.


- What problem does it solve?
 Laboratory samples are often hard to manage and can cause to problems during the workflow. Lab Sample Management System solves that problem by organizing and monitoring
sample data in one single app, so that allowed staff can view the samples safely, without making any effort.

- What does it actually do?
  1) Manage lab samples with full lifecycle tracking — from collection through processing to archival
  2) Full REST API with GET, POST, PUT, DELETE endpoints
  3) Input validation with meaningful error responses
  4) Persistent storage with PostgreSQL database
  5) Interactive API documentation via Swagger UI
 
- Why did I build this app?
  I built this project to apply backend development skills in a real-world domain. The pharma and biotech industry is heavily concentrated in the Darmstadt region — companies rely on reliable lab data management systems. This project simulates a simplified version of that workflow using Spring Boot and PostgreSQL.
